Testing the site

Prerequisites:

  • Install Rubygems

git clone git@github.com:redhat-cop/redhat-cop.github.io.git
cd redhat-cop.github.io
gem install bundler
bundle install
bundle exec jekyll serve

Once running, the site should be available at http://localhost:4000
